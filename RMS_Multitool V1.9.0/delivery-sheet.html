🤖 Setting up AI Document Extraction

This feature uses Anthropic's Claude AI to read your event documents and automatically fill in the fields below. You need a free Anthropic account and API key to use it.

1
Go to console.anthropic.com and create a free account or sign in.
2
In the Anthropic Console, navigate to API Keys and create a new key. Give it a name like RMS Multitool.
3
Copy the key — you will only see it once. Do not share it with anyone.
4
Click the RMS Multitool extension icon in your browser toolbar to open Settings, then paste the key into the Anthropic API Key field and save.
5
Come back here and upload your event document — Claude will extract the schedule, contacts and setup details automatically.

Anthropic offers free credits for new accounts. Usage costs are very low — a typical event document costs a fraction of a cent to process. Your API key is stored locally in the extension and is never shared.
